## 简介 ##

- 简单的 Web 项目开发骨架
- 实战gulp及其插件的功能

## 运行 ##

- 全局安装模块：npm install -g gulp
- 查看gulp版本：gulp -version
- 编写gulp配置文件：gulpfile.js
- 运行gulp任务：gulp [任务名称]
